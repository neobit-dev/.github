# Neobit (네오비트)

**농장·창고·축사 현장의 장비와 데이터를 하나로 잇는 IoT 플랫폼 기업입니다.**

2015년 한우 축산 플랫폼에서 출발해, 현장 제어·모니터링·생산관리를 모바일과 웹으로 통합 제공하는 **큐브팩토리(Cube Factory)** 를 운영하고 있습니다.

> 제어, 알람, 생산 기록, 시세까지 — 큐브팩토리로 농장·창고에 필요한 것을 **한곳에서.**

## Neobit이 하는 일

제조사마다 달라지는 앱과 장비를 **하나의 계정·하나의 플랫폼**으로 묶습니다. 이미 현장에 있는 장비를 그대로 두고 연결할 수 있습니다.

- **환경 제어** — 개폐기, 환기, 관수 등 자동·수동 제어
- **실시간 모니터링** — 센서·장비 상태 확인, 이상 알람
- **생산·운영** — 한우 관리, 작업 일정, 시세·방역 정보, 농장 운영 기록
- **CCTV** — 여러 브랜드 카메라를 한 화면에서
- **현장과 사무실** — 모바일 앱과 웹에서 같은 정보 확인

**농장·창고 운영자**와 **장비·설치 업체** 모두를 위한 앱과 관리 도구를 제공합니다.

## 큐브팩토리로 할 수 있는 것

| | |
|---|---|
| **통합 앱** | 개폐기, CCTV, 차량 인식, 생육 관리 등을 앱 하나로 |
| **다사업장** | 축사·하우스·창고를 한 아이디로 관리 |
| **자동화** | 시간, 센서, 일출·일몰, 날씨에 맞춘 환경 제어 |
| **알람** | 장비 이상·긴급 상황을 즉시 알림 |
| **축산** | 발정 알림, 번식·투약 일정, 개체 이력 |
| **구역 한눈에** | 동·하우스·창고별 상태를 지도처럼 확인 |
| **관리자 지원** | 고객·장비·설치·A/S 현황을 관리자 앱에서 |
